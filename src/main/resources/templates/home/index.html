Sobre Nós

Esse é um projeto da Gama Academy em parceria com a empresa M4U realizado pelo grupo 05 da turma 01. A ideia é criar uma plataforma de cadastro e login de usuários, construído através da linguagem Java. O front-and foi desenvolvido por bootstrap, HTML e CSS possuindo ligação com banco de dados através da ferramenta MySQL.
